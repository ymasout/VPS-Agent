# AI VPS 运维控制台：开发命令速查

以下命令默认在项目根目录 `E:\VPS-Agent` 下执行，终端使用 PowerShell。

## 1. 首次初始化

### 创建环境配置

```powershell
Copy-Item .env.example .env
```

### 创建并启用 Python 虚拟环境

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r apps\api\requirements-dev.txt
```

### 安装 Node.js 依赖

```powershell
pnpm install
```

### 检查开发工具

```powershell
python --version
node --version
pnpm --version
go version
docker version
docker compose version
```

## 2. Docker Compose 完整环境

### 首次构建并启动

```powershell
docker compose up --build
```

### 使用已有镜像启动

```powershell
docker compose up
```

### 后台启动

```powershell
docker compose up -d
```

### 查看服务状态

```powershell
docker compose ps
```

### 查看全部日志

```powershell
docker compose logs -f
```

### 查看单个服务日志

```powershell
docker compose logs -f web
docker compose logs -f api
docker compose logs -f agent
docker compose logs -f postgres
docker compose logs -f redis
```

### 停止并删除容器

```powershell
docker compose down
```

### 停止并删除容器及数据卷

此命令会清空本地 PostgreSQL 和 Redis 数据，仅用于需要完全重置开发环境时。

```powershell
docker compose down -v
```

### 重新构建单个服务

```powershell
docker compose build web
docker compose build api
docker compose build agent
```

### 不使用缓存重新构建

```powershell
docker compose build --no-cache
```

## 3. 本地开发

### 只启动 PostgreSQL 和 Redis

```powershell
docker compose up -d postgres redis
```

### 启动 FastAPI

先启用 Python 虚拟环境：

```powershell
.\.venv\Scripts\Activate.ps1
Set-Location apps\api
python -m uvicorn app.main:app --reload
```

API 默认地址：`http://localhost:8000`  
API 文档：`http://localhost:8000/docs`  
健康检查：`http://localhost:8000/healthz`

### 启动 Next.js

在另一个 PowerShell 窗口中执行：

```powershell
Set-Location E:\VPS-Agent
pnpm dev:web
```

Web 默认地址：`http://localhost:3000`

### 启动 Go Agent

确保 API 已经启动，然后在另一个 PowerShell 窗口执行：

```powershell
Set-Location E:\VPS-Agent\apps\agent
go run .\cmd\agent
```

### 退出 Python 虚拟环境

```powershell
deactivate
```

## 5. 常用接口检查

### 创建一次性 Agent 注册令牌

管理员令牌默认从 `.env` 的 `ADMIN_API_TOKEN` 读取。以下为开发环境示例：

```powershell
$headers = @{ "X-Admin-Token" = "local-admin-change-me" }
$body = @{ name = "prod-vps-01"; expires_in_minutes = 30 } | ConvertTo-Json
Invoke-RestMethod -Method Post `
  -Uri http://localhost:8000/api/v1/registration-tokens `
  -Headers $headers `
  -ContentType "application/json" `
  -Body $body
```

返回的 `token` 只应提供给目标 Agent，并且注册成功后不能再次使用。

### 执行数据库迁移

```powershell
Set-Location apps\api
..\..\.venv\Scripts\python.exe -m alembic upgrade head
Set-Location ..\..
```

### PowerShell 请求 API 健康状态

```powershell
Invoke-RestMethod http://localhost:8000/healthz
```

### 查看 Agent 心跳日志

```powershell
docker compose logs -f api agent
```

正常情况下，Agent 会按照配置的时间间隔发送心跳，API 日志中会出现 `agent.heartbeat`。


## 7. 日常推荐流程

完整容器开发：

```powershell
docker compose up
```

需要重新构建时：

```powershell
docker compose up --build
```

提交代码前：

```powershell
.\.venv\Scripts\python.exe -m pytest apps\api\tests
.\.venv\Scripts\python.exe -m ruff check apps\api
pnpm test:web
pnpm lint:web
pnpm build:web
Set-Location apps\agent
go test .\...
Set-Location ..\..
```

结束开发：

```powershell
docker compose down
```
